# Komal Analytics Portfolio

A modern, interactive portfolio showcasing analytics projects, dashboards, and reports.

## 📋 Overview

This portfolio demonstrates expertise in data analytics, visualization, and dashboard development. It features sample projects with interactive elements and responsive design.

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Dark Mode Support** - Eye-friendly dark theme with toggle functionality
- **Interactive Dashboards** - Click-based interactions and filtering capabilities
- **Smooth Animations** - Professional transitions and hover effects
- **Modern UI** - Card-based layout with professional styling
- **Mobile-Optimized** - Full functionality on all screen sizes

## 📁 File Structure

```
Komal-Analytics-Portfolio1/
├── index.html          # Main portfolio page
├── styles.css          # Professional styling and responsive design
├── script.js           # Interactive functionality
└── README.md           # This file
```

## 🚀 Getting Started

1. **Clone or download** this repository
2. **Open `index.html`** in your web browser
3. **Explore** the dashboards and projects

No build process or server required - it's ready to use!

## 💻 Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with flexbox/grid layouts
- **JavaScript (ES6+)** - Interactive features and functionality

## 🎨 Customization

### Styling
Edit `styles.css` to customize:
- Colors and theme
- Layout and spacing
- Font styles
- Responsive breakpoints

### Interactivity
Modify `script.js` to add:
- New event listeners
- Custom filters
- API integrations
- Additional animations

### Content
Update `index.html` to:
- Add your own dashboards
- Include project descriptions
- Link to external resources
- Customize navigation

## 🔧 Key Functions

- `toggleDarkMode()` - Toggle between light and dark themes
- `filterDashboards(category)` - Filter dashboards by category
- `toggleMobileMenu()` - Toggle mobile navigation
- `fetchDashboardData(url)` - Fetch data from APIs

## 📱 Responsive Breakpoints

- **Desktop**: 1024px and above
- **Tablet**: 768px - 1023px
- **Mobile**: Below 768px

## 💡 Tips

- Use data attributes on cards (`data-category`) for filtering
- Add links to your deployed dashboards
- Include project descriptions in card content
- Update social links in the footer
- Add Google Analytics for tracking
