### 2. Automation Showcase
Located in `/Automation-Showcase/`

- **generate_roe_report.py**  
  Python script that automates weekly due-date reporting:
  - Auto-detects input file format (raw vs macro)
  - Processes work orders, SLA logic, overdue days
  - Builds styled Excel report (pivot + data sheet)
  - Optional Outlook email dispatch

- **roe_automation_showcase.html**  
  Demo showcase document with:
  - Overview of automation pipeline
  - Technical architecture
  - Code excerpts (Python + Batch)
  - Business impact metrics
  - Setup instructions

- **run_report.bat** (optional)  
  Windows batch launcher:
  - Handles Python dependency installation
  - Resolves input file via 4-method fallback
  - Runs the Python script with `--send` flag
